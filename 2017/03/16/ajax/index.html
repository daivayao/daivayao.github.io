ajax 的原始步奏及原理

  1. 创建XMLHttprequest对象 if(window.XMLHttprequest){
    var xhr=new XMLHttprequest
    }esle{
    var xhr.ActiveXObject(“Microsoft”); //兼容I6的对象
    }

  2. 准备发送 open(get或post,地址，true或false)方法 里面有3个参数(1):请求方式为get或post (2):请求地址,(如果是get请求那么请求参数必须在url中传递,post参数却写在send()里并不要转码;如果是get请求那么必须转码,转码的方法写在地址后面用+连接地址encodeURL(参数);(3)异步和同步的标志位true :表示异步 false：表示同步

  3. 执行发送动作 xhr.send();如果是get 那么参数就为null;如果是post 首先创建一个请求头:xhr.setRequestHeader("Content-Type","application/x-www-form-urlencoded")然后执行xhr.send(这是请请求参数)

  4. 指定毁掉函数
    `xhr.onreadystatechange=function(){
    if(xhr.readyState=4){ //表示服务器还回的数据可以使用了,但这个数据不一定是正确的所以还有判断xhr.status

    if(xhr.status==200){//表示数据是正确的可以使用，不是200 都是错误的数据
    var data=xhr.responseText;//拿到请求的数据了
}

    }
    }`
    ###readyState的状态和status的值

    • readyState有5种状态0:表示初始化1：执行发送 2：服务器已完全还回数据 3：数据在解析 4：可以使用了
    • xhr.status有3种值 1:200数据正常 2:404没找到资源 3:505服务器错误；

jsonp的原理:动态创建script标签，

通过他的src的属性发送跨域请求，然后服务端响应回来的数据格式为函数调用【foo（）】，所以在发送请求必须先定义一个函数，并且函数的名字与参数传递的名字要一致
